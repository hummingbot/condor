# Backlog de mejoras — scope `frontend/`

Mejoras acotadas al frontend React/TS (`frontend/src/`). Numeración **independiente** del
backlog general (`improvements/{todo,done}/`): contador único `NNN` dentro de este scope,
contando `todo/` + `done/`, arrancando en `001` y sin reutilizar.

- `todo/` — pendientes, una por archivo.
- `done/` — implementadas, con commits anotados (las cierra `/ship-improvement`).

Generado por `/improvements @frontend/` (fan-out por dimensión + verificación adversarial).
Para implementar un item: `/ship-improvement <id>` (ej. `/ship-improvement SEC-016`).
